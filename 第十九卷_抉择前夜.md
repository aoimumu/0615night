# 第十九卷 抉择前夜

三月十二日。手术已经过去六天。

蛍活了下来。手术成功——心脏瓣膜被替换成了机械瓣膜。但术后出现了并发症：肺部感染，急性肾功能不全的前兆。她被留在ICU观察。

在六天里，她只短暂清醒过两次。一次喊了「お母さん」。一次看到瞬之后——什么也没说，只是看着他，像是确认了什么。然后又闭上了眼睛。

医生说——继续观察。情况不是最坏的，但不是最好的。

这段时间里，瞬每天从大学下课后来医院。先坐电车，再步行十分钟。走到医院楼下的时候，他会习惯性地抬头数到十二楼的窗户。不知道哪一个窗户是她的病房——但他总是数。

三月十二日晚。东京的夜空难得有几颗星。

桜井家。瞬的房间。

瞬坐在书桌前。面前摊着那本料理笔记。他翻到第5道——「エビの天ぷら」。蛍的字。铅笔写的。旁边还有一个小括号：「瞬君は多分、衣がうまくつけられないと思う。」

——瞬君大概会裹不好面衣。

他笑了一下。然后把笔记本合上。但手没有离开封面。他的手指在纸页边缘摩挲着，像是要擦掉什么看不见的东西。

手机震了一下。

陽菜：「周六有空吗。下午两点，新宿御苑。有重要的话想对你说。」

瞬看着这行字。

周六。三月十五日。三天后。

他知道那句话是什么。从台场摩天轮开始他就知道。便利店里、银杏树下、富士山顶、代々木公園、LIVEHOUSE后台——她每一次看向他的眼神，都在提前预演那句还没有说出口的话。

他之前没有让她说出来。因为他觉得自己还没有准备好。

而现在——

他翻来覆去地看了这句话三次。

然后回复：「好。」

几乎是瞬间。陽菜：「你答应了——」

「嗯。」

「那——周六下午两点。新宿御苑。正门。」

「好。」

他没有再说别的话。陽菜也没有再发消息。但他们之间的已读标记都亮着。像是两个人都握着手机，却谁都不想先放下。

同一时刻，高橋陽菜的房间。

陽菜把手机抱在胸前，仰躺在床上。

他答应了。他终于把那个「好」字打出来了。

她翻了个身，打开抽屉拿出那个塞满便签纸的小盒子。数了数——七张。第一张是代々木公園那天晚上写的：「如果有一天／你的眼里／不再只有她的身影／那我——／会告诉你。」最后一张是上周写的：「等她好了之后／我要说出来。」

她把七张便签全部排在被子上。每一张都看过一遍。

然后她给絵里发了一条LINE：「周六。新宿御苑。我要说了。」

絵里的电话直接打进来了。

「你真的要说了？」

「真的。」

「万一——」

「没有万一了。」陽菜握紧手机，「絵里。我知道你在担心什么。但是——我等不了了。不是不想等了。是不能等了。因为我发现，每次看到他难过的时候，我想安慰他。但我伸出手，又觉得没有立场。只能发LINE说——『一切都会好的』。那种感觉很——」

「手が届かない。」絵里替她说完了。

「……对。就是那种。」

絵里在电话那头沉默了几秒。然后她轻声说：「那你去吧。不管结果怎样——你还有我。」

陽菜弯起嘴角。挂了电话后，她从衣橱里挑了一件淡粉色的连衣裙，熨好后挂在门背后。周六穿。

然后她又给瞬发了一条LINE：「晚安。周六见。」

瞬：「嗯。おやすみ。」

她在被子里把手机屏幕贴在脸上。凉的，但心里很暖。

同一时刻。東京大学医学部附属病院。十二楼ICU。

蛍醒了。这一次清醒的时间比之前长。

医生做完检查后，允许家属探视。静子先进去——握着女儿的手说了好多话。蛍一直点头，偶尔说一两个音节。脸色还是很苍白。身上插着好几根透明的管子，仪器在旁边不停地响。但她说话时的表情，是平静的。

静子出来后，对瞬说：「她让你进去。」

瞬穿上隔离衣，戴上口罩，走进ICU。

蛍半躺在床上，头侧向他。她的嘴唇发干，但她没有要求水。只是看着他走近。

「瞬君。こんばんは。」她的声音比平时更轻——轻得像一页薄纸从书上剥离。

「こんばんは。」

瞬在床边的椅子上坐下。机器规律的「ピッ、ピッ」声填满了他们之间的沉默。

「手術は——うまくいったんだって。」蛍慢慢地说，「でも、まだ少し、体がついていかない。ごめんね。心配かけて。」

——听说手术成功了。但身体还没跟上。对不起，让你担心了。

「你不用道歉。」瞬说。

蛍轻轻摇了摇头。「謝りたいわけじゃない。言いたいのは、別のこと。」

——不是想道歉。想说的是别的。

瞬等她。机器声很有规律地响着。

「土曜日でしょ。」蛍说，「陽菜さんが、何か言うんでしょ。」

——是周六对吧。陽菜さん，要说什么事情对吧。

瞬的呼吸滞了一下。

「你怎么知道。」

「インスタグラム。陽菜さんの——今日、新宿御苑の写真をアップしてた。『週末、大事な日』って。それに——瞬君の返信の書き方が、いつもとちょっと違ってた。優しかった。」

——Ins。陽菜さん今天发了新宿御苑的照片。说「周末，重要的日子」。而且——瞬君回我消息的语气和平时有一点不一样。比较温柔。

她说完，闭了一下眼睛。机器的「ピッ」声像是替她说完了一句话。

「蛍。我——」

「言わなくていい。」蛍打断他，「あなたが何を選んでも、私は大丈夫。昔から——ずっと大丈夫だった。だから今回も大丈夫。だから——ちゃんと行って。ちゃんと聞いて。」

——不用说。不管你选什么我都没关系。从小开始就一直没关系。所以这次也没关系。所以——去吧。好好听她说。

机器还在响。但瞬听得见自己的心跳在加急。

「蛍。お前は——」

「うん？」

「お前は、俺に、もっと怒ってもいいんだ。」

——你可以对我更生气一些的。

蛍眨了眨眼。然后她笑了——是那种很淡很淡、几乎没有弧度的笑，但眼睛亮了一下。

「怒ってるよ。」她说，「怒ってる。すごく。小さいころからずっと。あなたが私を妹みたいにしか見てくれなかったこと。私がいくつになっても、あなたにとっては『向かいの家の蛍』だったこと。ぜんぶ、怒ってる。」

——生气啊。我很生气。从很小的时候开始就一直。气你一直只把我当妹妹看。气我不管长到几岁，对你来说都只是「对面那家的蛍」。全部都。

「でも——」她深吸了一口气，机器的警报没有响。「でも、それ以上に、ありがとうって思ってる。だから大丈夫。——私は大丈夫。」

——但是，比起那个，更觉得谢谢你。所以没关系。——我没关系。

这句话说完，她闭上了眼睛。不是昏过去了，而是像说完了所有的话之后，整个人从里到外都松弛了。

瞬站起来。他伸出手——本想去握她的手。但手在半空停住了。他想起自己的手上有隔离手套，手套外面不是他能碰到的皮肤。于是他把手收回来，放在床沿上。

「土曜日。ちゃんと行く。でも——ちゃんと帰ってくる。」

——周六。我会去。但是——我会回来。

蛍没有睁开眼睛。但她轻轻点了点头。

「うん。」她说，「帰ってきて。」

——嗯。回来吧。

晚上十点半。瞬走出医院。

电车还有一班。他站在站台上时，手机响了LINE。

陽菜：「睡了吗。」

「没。刚从医院出来。」

「这么晚去——蛍ちゃん怎么样了——」

「醒了。说了几句话。现在睡了。」

陽菜隔了几秒回：「那就好。瞬君，你累吗。」

「有一点。」

「那就早点休息。周六——如果累了可以不来的。延期也没关系——」

「去。」他回，「一定到。」

站台的风吹过来。很冷。他把手机揣进口袋。看着对面夜空中唯一可见的几颗星。

同一时刻。高橋陽菜在房间里，看着手机屏幕上那两个字——「一定到。」她把手机捂在胸口，閉上眼睛。

同一时刻。蛍躺在ICU的病床上，也闭着眼睛。她的嘴唇在轻微翕动，似乎在默念什么东西。如果凑得很近，也许能听见——她念的是他最后说的那句：「帰ってきて。」

三人在东京不同的角落，在这个三月十二日的夜晚，各自闭眼。

窗外的风从西高东低。吹过世田谷区的小巷，吹过新宿的高楼，吹过不忍池的薄冰。东京没有下雪。但这夜冷得像雪落在身上的前一秒。

**蛍的日记 — 三月十二日，深夜**

今天醒了。和瞬君说了话。

我告诉他：我很生气。

这大概是我这辈子对他说过最诚实的一句话。

诚实说话的代价是——胸腔里那颗新的机械瓣膜好像跳得比平时重了一点点。医生说它还没完全适应身体。也许它也不想适应。

但不管怎样，我把话说了。

然后我说：我没关系。

这句是谎话。

但没关系。

如果周六一切顺利——那周日我会发LINE给你。发什么呢。还没想好。大概会发——「今日、ちゃんと話せてよかったね。」——今天好好说出来了，真好呀。

就是这句吧。

晚安。瞬君。

晚安。陽菜さん。

晚安。新买的连衣裙，和新到的心脏瓣膜。

（第十九卷 完）

*从六十二跳到六十三。*
*只跳了一格。*
*但机器不懂。那一格，是「さようなら」。*
